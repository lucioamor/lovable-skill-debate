# Lovable Skill: Debate

Use this repository to import the `/debate` skill into [Lovable](https://lovable.dev).

## Import into Lovable

1. Open `Settings -> Skills -> Add -> Import from GitHub`.
2. Paste this repository URL:

```text
https://github.com/lucioamor/lovable-skill-debate
```

## What it does

`/debate` runs an advisory expert panel with opposing views before implementation. It is useful for product, UI, UX, copy, architecture, performance, data, and security decisions where the trade-offs matter.

The importable skill file is [`SKILL.md`](./SKILL.md).

## Version

Current version: [`v1.0.0`](./VERSION.md)

The skill prints its installed version, this source repo URL, and a version status in its output: current, update available, or unverified.
